# 使用方法

## 1. キーワード検索

サイドバーの「キーワード」欄に検索したい単語を入力してください。

- 複数キーワードはスペース区切りで入力可能です。
- 部分一致で検索されます。

## 2. 条件絞り込み

### **省庁の選択**

ツリー形式で省庁を選択できます。

- 外局がある場合、管轄省庁を選択すると全て選択されます。
- 個別に選択することも可能です。

### **カテゴリ・資料形式・対象年度の選択**

- 資料のカテゴリ、資料形式・対象年度から絞り込むことができます。
- 複数選択可能です

### 会議体の選択

- 会議資料に限り、会議体名を指定して選択することが可能です。
- 会議体選択欄にキーワードを入力することで、会議体を絞り込むことができます。

## 3. 検索結果

検索ボタンをクリックすると、条件に合致する資料が表示されます。

⚠️ 会議体を選択した場合、予算タブの検索は実行されません。

⚠️ 検索結果件数が多い場合、表示に時間がかかることがあります。

# 収録資料に関する情報

### カテゴリについて

下記の資料について収集・収録をしています。

| **カテゴリ** | **定義** | **収集対象年度** | **収録状況** |
| --- | --- | --- | --- |
| 予算:概算要求 | 各省庁が翌年度に必要と見込む予算額を政府（財務省）に「要求」する段階の文書 | 2013(平成25)年度以降 | 平成令和8年度分まで収集・格納済 |
| 予算:当初予算 | 当初予算（暫定予算含む） | 2013(平成25)年度以降 | 令和7年度分まで収集・格納済 |
| 予算:補正予算 | 補正予算 | 2013(平成25)年度以降 | 令和7年度分のうち2025年7月時点で公開されている分まで収集・格納済 |
| 会議資料:各種本部・連絡会議等 | 内閣官房が主体で設置している内閣主導の重要政策の諮問機関で、省庁横断的な性格を持つ会議体 | 2023(令和5)年度以降 | 一部を除き2025年9月時点で公開されている分 |
| 会議資料:審議会 | 法令に基づいて各省庁管轄の下設置された正式な諮問機関 | 2023(令和5)年度以降 | 次の省庁が管轄する審議会（外局含む）：内閣府、復興庁、総務省、文部科学省、厚生労働省、農林水産省、経済産業省、経済産業省、国土交通省 |

### 資料形式について

資料の形式について、下記の定義に沿って分類しております。

| **資料形式** | **定義（弊社独自）** |
| --- | --- |
| 予算:予算概要 | 予算の事業ごとの使用目的や予算規模などの概要を説明するもの。 |
| 予算:予算説明書 | 当該省庁の予算全体または各事業ごとの予算の背景や目的、要点について説明するもの |
| 予算:明細書 | 予算の項目ごとの金額等の明細が記載されたもの。 |
| 会議資料:議事録(概要・要旨) | 会議の参加者の発言など、議論の内容や経緯が記されたもののうち、概要や要旨を抜粋したもの。 |
| 会議資料:議事録(全文) | 会議の参加者の発言など、議論の内容や経緯が記されたもののうち、参加者の発言などの全文が記載されたもの。 |
| 会議資料:決定・答申・報告書・提言等 | 会議での審議結果や検討内容を踏まえ、方針・結論・提言などをまとめた最終的な成果資料。 |
| 会議資料:配布資料・議事次第・調査報告・意見書等 | 会議での審議や報告の際に配布・共有された資料や、参考情報として提出されたもの。 |
| 会議資料:設置根拠・運営規則・名簿等 | 会議の設置目的や運営方法を定めた資料、または構成員や関係者の名簿などをまとめたもの。 |
| 会議資料:その他 | 上記以外の会議関連資料。 |
